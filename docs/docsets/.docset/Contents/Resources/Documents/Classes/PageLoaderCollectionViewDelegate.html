PageLoaderCollectionViewDelegate

open class PageLoaderCollectionViewDelegate<CellType: UICollectionViewCell, Type: Equatable>: CollectionViewBlockDelegate<CellType, Type>
    where CellType: Configurable

Basic UICollectionViewDelegate abstracting paging. When the CollectionView is scrolled to the bottom, will check if should fetch the next page.
